# 03 PC端原型设计说明 — 管理层数据大盘+工单调度

> 基于设计系统规范与PRD场景定义，输出管理层PC端的3个核心页面原型说明：数据大盘首页、工单调度页、巡检质量分析页。所有页面面向物业经理/管理层角色，聚焦"管控"场景，信息密度高，支持数据下钻与空间可视化。

---

## 页面一：数据大盘首页

### 1.1 页面概述

| 项目 | 说明 |
|------|------|
| **页面名称** | 数据大盘首页 |
| **用途** | 管理层一屏掌握园区巡检全局态势，核心KPI、空间分布、趋势变化、实时动态集中呈现 |
| **面向角色** | 物业经理（王丽华）、管理层决策者 |
| **使用频率** | 高频，每日晨会/日常监控/决策前首选入口 |

### 1.2 布局结构

整体采用 **"顶部筛选栏 + 三栏主内容区 + 底部快捷操作"** 的PC大屏布局：

```

├─────────────────────────────────────────────────────────────────────────────┤
│ ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐                     │
│ │ KPI-巡检  │  │ KPI-隐患  │  │ KPI-工单  │  │ KPI-覆盖  │   ← 关键指标卡片区    │
│ │  完成率   │  │   数量    │  │  待处理   │  │   率     │                     │
│ └──────────┘  └──────────┘  └──────────┘  └──────────┘                     │
├──────────────┬──────────────────────────────────────────────┬───────────────┤
│              │                                              │               │
│   左侧面板    │              中部 GIS 地图区                   │   右侧面板     │
│  [240px宽]   │           [自适应宽度, 最小500px]              │  [320px宽]    │
│              │                                              │               │
│  近7日隐患   │   • 园区热力图叠加层                          │  设备健康度    │

│  巡检效率    │   • 轨迹线（已巡实线/当前绿线/规划虚线）       │  巡检效率对比   │
│  柱状对比图   │                                              │  分组柱状图     │
│              │   [地图工具栏: 缩放/图层切换/全屏/测距]         │               │

```

### 1.3 关键元素清单

#### 顶部筛选栏

| 元素 | 类型 | 说明 |
|------|------|------|
| 面包屑导航 | 文本链式 | "数据大盘 / 首页"，当前页无链接 |
| 日期筛选器 | 日期选择器 | 默认"今日"，快捷选项：今日/本周/本月/自定义范围 |
| 园区切换 | 下拉选择器 | [V2.0] 当前禁用，显示"科技园A区"，预留多园区入口 |
| 告警提醒 | 图标按钮 | 🔔 铃铛图标，未读红点+数字徽标；点击展开告警消息浮层 |

#### 关键指标卡片区（4列等分）

| 卡片 | 核心指标 | 辅助信息 | 状态色 |
|------|---------|---------|--------|
| 巡检完成率 | 40px 数据大字，如"87.5%" | 环比变化箭头+百分比，今日已完成/计划总数 | 科技蓝-500 |
| 隐患数量 | 40px 数据大字，如"23" | 较昨日+5，已闭环/待处理/新上报分布 | 错误红（隐患数>阈值时变红） |
| 待处理工单 | 40px 数据大字，如"8" | P0紧急/P1严重/P2一般分级数量 | 警告黄（>5件时变黄） |
| 覆盖率 | 40px 数据大字，如"92.3%" | 覆盖面积/总面积，漏检区域数量 | 成功绿 |

> 每张卡片：白色背景卡片（圆角12px），顶部4px彩色条区分状态，卡片内边距20px，悬停时显示"查看详情"文字按钮。

#### 中部GIS地图区

| 元素 | 类型 | 说明 |
|------|------|------|
| 园区底图 | GIS地图 | 2D矢量园区地图，支持缩放/拖拽/旋转 |
| 热力图叠加层 | 地图图层 | DBSCAN聚类结果可视化，低(#FEF3C7)/中(#F59E0B)/高(#EF4444)三色渐变 |
| 问题点位标记 | Marker集群 | P0(错误红)/P1(警告黄)/P2(信息蓝)水滴形标记，数字徽标显示该点位工单数 |
| 实时巡检员位置 | 脉冲定位点 | 科技蓝-500圆点+扩散动画环，显示巡检员姓名缩写 |
| 轨迹线 | 折线 | 已巡实线(#2563EB 3px)/当前绿线(#10B981)/规划虚线(#9CA3AF) |
| 地图工具栏 | 图标按钮组 | 缩放±/图层切换/全屏/测距，白色圆形底板40px带投影 |
| POI聚合点 | 聚合Marker | 地图缩小时同类点位自动聚合，圆形面积与数量成正比 |

#### 左侧面板 — 趋势图表

| 元素 | 类型 | 说明 |
|------|------|------|
| 近7日隐患趋势 | 折线图 | X轴日期，Y轴隐患数量，1-2条线（上报数/闭环数），科技蓝/成功绿，带数据点悬停 |
| 巡检效率对比 | 柱状图 | 按区域/人员分组，圆角柱，显示完成率或平均巡检时长 |

#### 右侧面板 — 状态图表

| 元素 | 类型 | 说明 |
|------|------|------|
| 设备健康度 | 环形图（甜甜圈） | 中心显示整体健康度百分比，分段：正常(成功绿)/预警(警告黄)/故障(错误红) |
| 巡检员效率对比 | 分组柱状图 | 各巡检员完成率对比，悬停显示具体数值 |

#### 底部快捷操作栏

| 按钮 | 类型 | 说明 |
|------|------|------|
| 生成日报 | 次按钮 | 点击生成当日巡检日报PDF，下载至本地 |
| 导出数据 | 次按钮 | 导出当前筛选条件下的原始数据Excel |
| 快速派单 | 主按钮 | 跳转至工单调度页，并自动筛选待分派工单 |

### 1.4 交互逻辑

| 交互 | 触发 | 结果 |
|------|------|------|
| 指标卡点击 | 点击任一KPI卡片 | 跳转至对应详情页（如"待处理工单"跳转工单调度页并带筛选条件） |
| 指标卡悬停 | 鼠标悬停 | 卡片阴影加深(Shadow-S→Shadow-M)，显示"查看详情→"文字按钮 |
| 地图点击钻取 | 点击问题点位Marker | 弹出地图弹窗卡片（底部三角指向POI），展示点位名称/问题类型/工单数/最新状态 |
| 地图聚合点点击 | 点击聚合Marker | 地图自动放大至该簇点位展开层级 |
| 图表悬停详情 | 悬停折线数据点/柱形 | 浮层显示精确数值+日期/区域/人员信息 |
| 日期筛选切换 | 选择不同日期范围 | 全页数据刷新（KPI+地图+图表同步更新），加载态显示骨架屏 |
| 地图图层切换 | 点击图层按钮 | 切换显示：全部/仅P0/仅P1/仅P2/热力图/轨迹线 |
| 告警提醒点击 | 点击🔔图标 | 右侧滑出消息抽屉，展示未读告警列表（P0优先置顶） |

### 1.5 AI/GIS嵌入点标注

| 能力 | 嵌入位置 | 具体表现 |
|------|---------|---------|
| **AI-趋势预测** | 左侧面板"近7日隐患趋势" | 折线图延伸虚线显示未来3天预测趋势，标注"AI预测"图例 |
| **AI-异常检测** | KPI卡片"隐患数量" | 当隐患数突增>50%时，卡片顶部色条变红+右上角"⚠️异常"标签 |
| **AI-设备健康评分** | 右侧面板"设备健康度" | 环形图数据由AI综合设备状态+历史故障率+运行参数计算得出 |
| **GIS-热力图聚类** | 中部地图 | DBSCAN/OPTICS算法聚类结果叠加，识别问题高发区域 |
| **GIS-实时定位** | 中部地图脉冲点 | WebSocket推送巡检员GPS坐标，10秒刷新一次 |
| **GIS-轨迹回放** | 中部地图 | 支持选择巡检员+日期，播放该日巡检轨迹动画 |
| **GIS-覆盖率计算** | KPI卡片"覆盖率" | GIS叠加巡检轨迹与园区边界，自动计算空间覆盖百分比 |

### 2.3 关键元素清单

#### 顶部筛选栏

| 元素 | 类型 | 说明 |
|------|------|------|
| 面包屑 | 文本链式 | "工单调度" |
| 搜索框 | 文本输入 | 按工单编号/问题描述/上报人搜索，支持模糊匹配 |
| 新建工单 | 主按钮 | 管理层手动创建工单（应急场景），跳转新建弹窗 |
| 告警提醒 | 图标按钮 | 同数据大盘页 |

#### 左侧面板 — 筛选器+隐患列表

**筛选器栏（筛选栏高度48px，折叠展开）：**

| 筛选项 | 类型 | 选项 |
|--------|------|------|
| 状态 | 多选标签 | 待审核/已派单/处理中/待验收/已闭环/已退回 |
| 优先级 | 多选标签 | P0(错误红)/P1(警告黄)/P2(信息蓝)/P3(成功绿) |
| 问题类型 | 下拉选择 | 消防/电气/结构/设备/环境卫生/其他 |
| 时间范围 | 日期选择器 | 今日/近7天/近30天/自定义 |
| 排序 | 下拉选择 | 按时间倒序(默认)/按优先级/按状态 |

**隐患列表（工单卡片式）：**

| 元素 | 说明 |
|------|------|
| 工单卡片 | 白色背景卡片，左侧4px色条标识优先级(红/橙/黄/蓝)，内边距16px |
| 卡片内容 | 工单编号(如 WO-20260427-008) + 问题类型标签 + 状态标签 + 上报时间 + 摘要描述(截断2行) |
| 选中态 | 选中卡片边框变为科技蓝-500，左侧色条加宽至6px |
| 分页器 | 每页10条，底部居中分页控件 |

#### 中部详情面板

**工单头部信息：**

| 元素 | 说明 |
|------|------|
| 工单编号 | H2字号，如"WO-20260427-008" |
| 状态标签 | 状态色标签（待审核-信息蓝/已派单-科技蓝/处理中-警告黄/待验收-紫色/已闭环-成功绿/已退回-错误红） |
| 优先级标签 | P0-P3标签，颜色递进 |
| 创建时间 | 辅助文字，精确到分钟 |
| 上报人 | 巡检员姓名+头像缩略图 |

**问题描述区：**

| 元素 | 说明 |
|------|------|
| 问题描述 | 正文文字，展示巡检员上报的文字描述/NLP结构化结果 |
| 照片轮播 | 水平滚动，缩略图列表，点击放大查看；处置前后照片分Tab展示 |
| AI分级结果 | 信息卡片，显示"AI判定：P1严重 - 消防通道堵塞"，附带置信度百分比条(≥85%为绿色，<85%为黄色) |
| AI识别详情 | 展开面板，显示AI图像识别的具体标签（如"消防通道堵塞：置信度92%"、"杂物堆积：置信度88%"） |

**位置地图（Mini GIS）：**

| 元素 | 说明 |
|------|------|
| 地图容器 | 圆角12px卡片，高度200px，内嵌简版GIS |
| 问题点位 | 红色水滴Marker居中，标注问题类型图标 |
| 地址描述 | 地图下方文字："A区3号楼2层东侧消防通道"（自动从GIS坐标反解析） |
| 查看大图 | 文字按钮"在地图中查看"，跳转数据大盘并定位该点位 |

**处置方案推荐（AI）：**

| 元素 | 说明 |
|------|------|
| 方案卡片 | 浅色背景卡片，标题"AI推荐处置方案" |
| 方案内容 | 历史相似工单的处置步骤摘要（如"1.清理杂物→2.恢复通道→3.拍照确认"） |
| 参考工单 | 显示"参考工单 WO-20260420-003，处置时长15min" |
| 平均耗时 | 同类问题历史平均处置时长 |

#### 右侧面板 — 派单操作区

| 元素 | 说明 |

#### 底部状态流转图

| 元素 | 说明 |
